== Description ==

Logs sent emails related to orders, and displays them in a table on the order screen.

This plugin lets you keep track of what emails have been sent for each order. On each order screen, you will see a table with:
Sent Date
Who the email was sent to
The email subject line

We also have a premium plugin which offers the following additional features:

= View Email Content =
View the content of all logged emails sent

= Resend Emails =
Re-send any of the logged emails. You can also specify any email address to send to, incase the customer wants to use an alternative inbox.

= Lifetime updates =

Any updates to the premium plugin will be available to you at no extra cost.

= How to get the premium plugin =

Simply install this free plugin, and go to the settings within the WooCommerce Integrations tab to purchase a license key.

= Note - this plugin only works with WooCommerce. = 

Note - The email logs will only start from the moment you activate this plugin. Any orders made before this date will not show any email logs, even though your website may have sent emails.

== Frequently Asked Questions ==

How can I see the email log for an order?
On the order screen, this plugin adds a table called 'Order Emails Sent Log', displaying emails that have been sent.

What data can I see?
Sent date, 
who the email went to, 
the email subject, 
the email content (coming soon).

Does the plugin show me historic orders?
The email logs will only start from the moment you activate this plugin. Any orders made before this will not show any email logs, even though your website may have sent emails.

== Installation ==

1. Upload this plugin to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
