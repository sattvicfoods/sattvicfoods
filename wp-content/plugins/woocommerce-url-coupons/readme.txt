=== WooCommerce URL Coupons ===
Author: skyverge
Tags: woocommerce
Requires at least: 4.1
Tested up to: 4.8
WC requires at least: 2.5.5
WC tested up to: 3.1.0

Create a unique url that applies a discount and optionally adds one or more products to the customer's cart.

See http://docs.woothemes.com/document/url-coupons/ for full documentation.

== Installation ==

1. Upload the entire 'woocommerce-url-coupons' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
